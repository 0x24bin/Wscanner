# WsScanner

A another sqli scanner,maybe is a full scanner.

# About this project

写这个扫描器的时候还很年轻，觉得写出来一定特别牛逼，但后来由于工作和个人的各种原因，没有再写这个项目。

后来看了很多的扫描器案例以及自己的视野越来越宽，有了更好的想法，另外再返回来看的时候，对自己写的这些代码特别不满意。

故该项目为废弃的半成品，仅供后生研究使用，当然也欢迎各位小伙伴继续开发，无版权限制 :)
